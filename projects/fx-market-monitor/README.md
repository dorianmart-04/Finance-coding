# FX Market Monitor

## Objective
Build a Python-based monitoring tool for major FX pairs to analyze market behavior, volatility dynamics, and return distributions.

## Market Scope
- EUR/USD
- GBP/USD
- USD/JPY

## Methodology
- Download historical FX data
- Compute daily returns
- Estimate volatility and rolling statistics
- Analyze distributional properties
- Visualize key metrics for market monitoring

## Technologies
- Python
- pandas, numpy
- matplotlib
- yfinance

## Outputs
- Time series of FX prices and returns
- Volatility estimates
- Return distribution histograms

## Next Steps
- Extend to rates and credit instruments
- Add alerting features for volatility/spread shocks
- Integrate scenario analysis
